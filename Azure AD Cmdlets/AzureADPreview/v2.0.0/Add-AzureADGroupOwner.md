# Add-AzureADGroupOwner

## SYNOPSIS
Add an owner to a group

## SYNTAX

```
Add-AzureADGroupOwner -ObjectId <String> -RefObjectId <String> [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

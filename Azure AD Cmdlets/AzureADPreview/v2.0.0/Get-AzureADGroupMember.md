# Get-AzureADGroupMember

## SYNOPSIS

## SYNTAX

```
Get-AzureADGroupMember -ObjectId <String> [-Top <Int32>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

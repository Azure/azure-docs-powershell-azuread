# Remove-AzureADDevice

## SYNOPSIS
Deletes a specific device in Azure Active Directory

## SYNTAX

```
Remove-AzureADDevice -ObjectId <String>
```

## DESCRIPTION

## EXAMPLES

### Remove a device from the directory
```
Remove-AzureADDevice -ObjectId 99a1915d-298f-42d1-93ae-71646b85e2fa
```

## PARAMETERS

### -ObjectId
The unique identifier of a device in Azure Active Directory (ObjectId)

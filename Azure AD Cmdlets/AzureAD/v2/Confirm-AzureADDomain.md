# Confirm-AzureADDomain

## SYNOPSIS
Validate the ownership of a domain.

## SYNTAX

```
Confirm-AzureADDomain -Name <String> [<CommonParameters>]
```

## DESCRIPTION
The **Confirm-AzureADDomain** cmdlet validates the ownership of an Azure Active Directory domain.

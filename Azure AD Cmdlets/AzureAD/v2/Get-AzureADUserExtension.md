# Get-AzureADUserExtension

## SYNOPSIS
Gets a user extension.

## SYNTAX

```
Get-AzureADUserExtension -ObjectId <String> [<CommonParameters>]
```

## DESCRIPTION
The **Get-AzureADUserExtension** cmdlet gets a user extension in Azure Active Directory (AD).

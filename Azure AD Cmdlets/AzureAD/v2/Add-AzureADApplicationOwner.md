# Add-AzureADApplicationOwner

## SYNOPSIS
Add an owner to an application

## SYNTAX

```
Add-AzureADApplicationOwner -ObjectId <String> -RefObjectId <String> [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```

```
